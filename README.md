# Water Bottle E-commerce (Frontend)

Modern, responsive multi-page frontend demo for a Water Bottle E-commerce platform.

## Pages
- `index.html` (Home)
- `products.html` (Products + filters)
- `product.html` (Product details)
- `cart.html` (Cart)
- `checkout.html` (Checkout UI)
- `about.html` (About)
- `contact.html` (Contact)







## How to run
Open `index.html` in a browser.

Tip: For the best experience (and to avoid browser restrictions), serve the folder with a simple local server:
- VS Code: Live Server extension
- Python: `python -m http.server 5500 --bind 127.0.0.1`
- npm (no install needed): `npm run start`

## Notes
- Cart is stored in `localStorage` (frontend-only).
- Payments are UI-only (no backend).
- Includes dark mode toggle, mobile menu, and a subtle animated water background.
